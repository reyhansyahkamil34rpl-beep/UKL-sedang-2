# UKL-sedang-2
Deskripsi Program

Program ini dibuat untuk menghitung volume tabung berdasarkan jari-jari dan tinggi yang dimasukkan oleh pengguna.
Rumus yang digunakan untuk menghitung volume tabung adalah:
Dimana:
V = Volume tabung
r = Jari-jari alas tabung
t = Tinggi tabung
π (Pi) = Konstanta ≈ 3,14159 (menggunakan Math.PI di Java, lebih akurat)
Fitur Program
Input jari-jari dan tinggi dari pengguna
Menggunakan fungsi/metode khusus untuk perhitungan volume
Output berupa volume tabung dalam satuan cm³
Struktur program lebih terorganisir dengan method hitungVolume()
Teknologi & Konsep yang Digunakan
Konsep	Penjelasan
Java	Bahasa pemrograman yang digunakan
Scanner	Untuk menerima input dari keyboard
Method (Fungsi)	Struktur modular untuk menghitung volume
Math.PI	Konstanta π yang disediakan oleh Java
Cara Menjalankan Program
Buka compiler/IDE Java
Compile program:
javac UKLsedang2.java
Jalankan program:
java UKLsedang2
Masukkan jari-jari dan tinggi tabung sesuai permintaan
 Contoh Output Program
--- PROGRAM MENGHITUNG VOLUME TABUNG ---
Masukkan jari-jari tabung (Cm): 7
Masukkan tinggi tabung (Cm): 10

--- HASIL ---
Volume tabung: 1539.3804002589986 Cm³
Kesimpulan
Program UKLsedang2 sederhana namun efektif untuk mempelajari:
Konsep perhitungan matematika pada Java
Pemanfaatan method sendiri
Input dan output data pengguna
Penerapan konstanta Math.PI untuk perhitungan geometri
